\#BW Custom Sidebar Blocks allows you to place blocks of post content on your Wordpress site as widgets.

== Description ==

\#BW Custom Sidebar Blocks creates a custom post type and a widget to insert post content into widgetized areas.

== Installation ==

1. Upload bw-csb folder to your /wp-content/plugins/ directory
2. Activate the plugin in Wordpress admin
3. Insert post content into widgetized areas  
	3a. Create a new "Sidebars" post, and set link in sidebar
	3b. Insert the "#BW - Custom Sidebar Block" widget into a sidebar and select the post in the widget options
